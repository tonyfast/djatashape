Overview

# (void) transform(transformers)

Transform adds named columns to the table are defined by cursors and a function fn.

Examples:

Create two new columns mean and std. 

table.transform
  mean: { cursors: ['x','y'], fn: (x,y)-> (x+y)/2 }
  std: { cursors: ['x','y'], fn: require('d3').deviation }

# (void) filter(columns..., fn)

Filter elements columns based on a predicate function.

Examples:

Filter columns ``x`` and ``y`` 

table.filter 'x','y', (x,y)-> x > 0 and y < 5

Parameters:

  • columns ( String ) — a list of columns to include in the predicate function
  • fn ( Function ) — a predicate function with access to each of the columns.

# (void) concat(new_values)

Concatenate new values to the table. append in the column direction or row direction, respectively.

Examples:

Add a Two Rows 

table.concat
  values: [
    [-3,4]
    [1,9]
  ]

Add One Column. The Array has a length of six because two rows were just added.

  columns:
    z: [-3,4,1,9,6,-4]
